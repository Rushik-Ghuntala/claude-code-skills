Usage: `/x-thread [topic + key points]`

Examples:
- `/x-thread How to optimize React performance — memo, useMemo, lazy loading, code splitting`
- `/x-thread My journey from bootcamp to senior engineer in 3 years`
- `/x-thread 7 Git commands every developer should know`
- `/x-thread Why we mass-migrated from microservices to a monolith`
