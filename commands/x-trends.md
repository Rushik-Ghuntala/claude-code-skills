Invoke the `trend-discovery` skill with the user's specified niche.

Usage: `/x-trends [niche]`

Examples:
- `/x-trends` — scan all dev topics
- `/x-trends React` — focus on React ecosystem trends
- `/x-trends AI coding tools` — focus on AI-assisted development
- `/x-trends DevOps` — focus on infrastructure and deployment trends
