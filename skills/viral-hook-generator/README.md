# Viral Hook Generator

> Generate scroll-stopping tweet openers using 15 proven hook archetypes adapted for developer audiences.

## Usage

```
/x-hook [topic]
```

## What It Does

Generates 5-7 hook variations for any developer topic using 15 battle-tested archetypes (Contrarian, Curiosity Gap, Pain Point, Data Lead, etc.). Each hook is scored on Curiosity (0-10), Specificity (0-10), and Algorithm-Friendliness (0-10), then ranked by combined score.

This is the **foundational skill** — its hook patterns are used by post-creator, thread-composer, article-generator, and content-calendar.

## Example

```
/x-hook "React Server Components"

1. [Score: 9.2] Contrarian
   "Unpopular opinion: React Server Components
   are the biggest shift since hooks — and most
   teams are using them wrong."

2. [Score: 8.7] Data Lead
   "I benchmarked React Server Components on 12
   production apps. Average bundle size dropped 43%.
   But that's not the interesting part."

3. [Score: 8.5] Pain Point
   "The #1 mistake devs make with React Server
   Components: treating them like regular components.
   Here's why that costs you 200ms per page load."
```

None — this is a foundational skill used by others.
