# Thread Formatting Rules

## Character Limits
- **Hard limit**: 280 characters per tweet
- **Optimal range**: 200-250 characters (long enough for substance, short enough for readability)
- **Hook tweet**: Can be shorter (140-200) for impact
- **CTA tweet**: 150-250 characters

## Whitespace & Line Breaks
- Use 1 blank line between distinct thoughts within a tweet
- Short lines are easier to read than long paragraphs
- Break after key phrases for emphasis:
  ```
  "I mass-deleted 40K lines of code today.

  Tests still pass.

  Best refactor of my career."
  ```
- Max 4-5 lines per tweet on mobile display

## Emoji Usage
- Use sparingly (0-2 per tweet)
- Best uses: bullet points, section markers, emphasis on key metrics
- Avoid: random decorative emoji, emoji-heavy sentences
- Thread numbering: "1/" is cleaner than emoji numbers

## Numbering Convention
- Use "1/" format (not "1.", "(1)", or "Tweet 1")
- This is the standard X thread convention
- Place at the start of each tweet
- Exception: Hook tweet can omit the number for cleaner presentation

## Bridge Sentences
Connect tweets with transitional phrases:
- "But here's the thing..." (pivot)
- "And it gets better..." (escalation)
- "The result?" (payoff setup)
- "Here's what most people miss:" (insider knowledge)
- Avoid: starting every tweet the same way

## Hook Tweet Rules
- NO links (algorithmic penalty)
- NO hashtags (looks spammy in hooks)
- Maximum curiosity gap
- Specific numbers when possible
- Must make the reader NEED to read tweet 2
- End with ":" or "👇" to indicate thread follows

## Final Tweet (CTA) Rules
- Summarize the key takeaway in 1 sentence
- Include a clear call-to-action
- Options: "Follow for more [topic]", "Bookmark this thread", "What would you add?"
- Can include "🧵 If you enjoyed this thread:" prefix
- Do NOT place links here (use self-reply)

## Self-Reply Link Placement
- Post a reply to your own thread with:
  - Links to resources mentioned
  - Your newsletter/project link
  - "If you enjoyed this, I write about [topic] weekly. Follow + 🔔"
- Self-replies avoid the algorithmic link penalty in the main thread

## Media Placement Strategy
- **Tweet 1**: No media (text hooks perform better)
- **Tweets 2-5**: Code screenshots or diagrams work well here
- **Tutorial threads**: Every step should have a code screenshot
- **Final tweet**: Optional summary graphic/infographic
- Media gets a 2x algorithmic boost — use it strategically

## Thread Length Guidelines
| Length | Best For | Drop-off Risk |
|--------|----------|---------------|
| 3-5 tweets | Quick tips, hot take expansion | Low |
| 5-8 tweets | Standard threads, comparisons | Low-Medium |
| 8-12 tweets | Tutorials, case studies | Medium |
| 12-15 tweets | Deep dives, comprehensive guides | High (front-load value) |
| 15+ tweets | Avoid — split into multiple threads | Very High |
